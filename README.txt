Shortcode attribute descriptions:
init_output - Initial gallery items output. Default: 12.
item_fetch - Number of items to fetch when pressing load more button. Default: 5.
columns - Layout the gallery in defined columns. Default: 3.

Plugin is using the jQuery lightGallery JS lib (version 1.4.0): https://github.com/sachinchoolur/lightGallery

Folders used for the jQuery lightGallery are "dist" and "lib" directories.
